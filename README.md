Para ter acesso, o usuário deve baixar os arquivos que estão no GitHub, ter instalado na sua máquina o Laragon e Git Bash. No Laragon deve iniciar em Start e no Prompt de Comando do Git Bash o usuário deve colocar: php artisan serve. Em seguida por a url: http://mathx.test/ que abrirá o site para uso.
